## 一、持仓概览

### 1.1 单只基金持仓明细

| 指标名称 | 指标说明 | 量化方法 | HTML输出方式 |
|----------|----------|----------|-------------|
| 基金代码 | 基金唯一标识 | 直接取输入数据 | 表格列，文本 |
| 基金名称 | 基金简称 | 直接取输入数据 / API `fund_info.name` | 表格列，文本（过长截断+title悬浮） |
| 基金类型 | 二级分类（混合型/QDII/股票型/指数型/货币型等） | API `fund_info.type` | 表格列，标签徽章 |
| 持仓市值 | 基金当前市值 = 最新份额 × 最新净值 | `shares × current_nav` | 表格列，货币格式 `¥xx.xx万` |
| 持仓成本 | 累计确认金额 | 交易记录逐笔累加 | 表格列，货币格式 |
| 盈亏金额 | 持仓市值 - 持仓成本 | `market_value - cost` | 表格列，红涨绿跌着色 |
| 盈亏率 | (市值 - 成本) / 成本 × 100% | `(market_value - cost) / cost` | 表格列，百分比，红涨绿跌 |
| 持仓权重 | 该基金市值 / 账户总市值 | `market_value / total_market_value` | 表格列，百分比 |
| 最新净值 | 基金最新单位净值 | API `fund_info.nav` | 表格列，数值 |
| 持仓份额 | 当前持有份额 | 交易记录计算（申购加、赎回减） | 表格列，数值 |
| 基金经理 | 现任基金经理姓名 | API `fund_info.manager` | 表格列，文本 |
| 综合评分 | 基金量化评价综合得分 | API `fund_evaluate.score` | 表格列，数值+等级徽章 |
| 操作建议 | 保留/观察/替换 | 基于评分+等级判断 | 表格列，标签 |

**HTML实现**: KPI 4卡片（总市值/总成本/总盈亏/收益率）+ 持仓分布饼图(ECharts donut) + 持仓明细表格（可横向滚动）

### 1.2 组合整体 KPI

| 指标名称 | 指标说明 | 量化方法 | HTML输出方式 |
|----------|----------|----------|-------------|
| 基金数量 | 当前持仓基金总数 | `len(holdings)` | KPI卡片，整数 |
| 总市值 | 所有持仓基金市值之和 | `Σ(market_value)` | KPI卡片，货币格式 |
| 总成本 | 所有持仓基金成本之和 | `Σ(cost)` | KPI卡片，货币格式 |
| 总盈亏 | 总市值 - 总成本 | `total_mv - total_cost` | KPI卡片，红涨绿跌 |
| 总收益率 | 总盈亏 / 总成本 | `(total_mv - total_cost) / total_cost` | KPI卡片，百分比 |
| 集中度预警 | 单只基金权重 > 20% 或基金数 > 12 | `max(weight) > 0.20 ∥ count > 12` | 警告卡片，橙色左边框 |

> **阈值设定依据**：
> - **单只基金权重 20%**：参考公募基金"双十限制"（单只证券不超过基金净资产的 10%），个人投资组合适当放宽至 20%，超过此比例则集中度风险显著上升
> - **基金数量上限 12 只**：基于分散化投资研究，持有超过 12 只基金的组合，边际分散化效益递减，同时增加管理复杂度和持仓重叠风险

| 交易统计 | 申购/赎回/分红次数和金额 | 交易记录逐笔统计 | KPI 3卡片组 |
| 总手续费 | 累计手续费 | `Σ(fee)` | 数值 |
| 费率 | 手续费 / 总申购金额 | `total_fee / total_subscribe` | 百分比 |
| 已实现盈亏 | 赎回金额 - 已清仓基金买入成本 | `total_redeem - total_liquidated_cost` | 数值 |
| 换手率 | 已清仓基金数 / 当前持仓数 | `liquidated_count / holding_count` | 百分比 |
| 投资年限 | 首次交易至最近交易的年数 | `(last_date - first_date).days / 365.25` | 数值（年） |
| 已清仓基金 | 已全部赎回的基金列表 | 份额 ≤ 0 的基金 | 列表（基金名称+清仓日期） |

## 二、收益风险表现

### 2.1 组合净值曲线 VS 对比指数

| 指标名称 | 指标说明 | 量化方法 | HTML输出方式 |
|----------|----------|----------|-------------|
| 组合净值曲线 | 账户每日组合净值走势 | `组合净值(t) = Σ(基金i权重 × 基金i净值(t) / 基金i初始净值)`，标准化起始值=1 | ECharts折线图 |
| 对比指数走势 | 偏股混合型基金指数 / QDII指数 / 沪深300 / MSCI全球 | 外部指数数据，标准化起始值=1 | 同一折线图中叠加 |

**对比指数选择规则**（PDF报告说明）:
- 偏股混合型基金指数（885001.WI）
- 债券型基金总指数（885005.WI）
- QDII股票型基金指数（885065.WI）
- QDII混合型基金指数（885067.WI）
- 根据持仓特征确定对比指数

**HTML实现**: ECharts双线折线图（组合净值 + 对比指数），X轴日期，Y轴标准化净值

### 2.2 收益风险对比（多期）

| 指标名称 | 指标说明 | 量化方法 | HTML输出方式 |
|----------|----------|----------|-------------|
| 累计收益率 | 买入以来总收益 | `(最终净值 / 初始净值) - 1` | KPI卡片 / 多期柱状图 |
| 年化收益率 (CAGR) | 复合年化增长 | `(1 + 累计)^(252/天数) - 1` | KPI卡片 |
| 近1月/3月/6月/1年/2年/3年收益 | 多期区间收益 | `(当前净值 / N个交易日前净值) - 1` | ECharts柱状图，红涨绿跌 |
| 成立以来收益 | 首笔净值至最新 | `(最新净值 / 首个净值) - 1` | 柱状图中的一项 |
| 年化波动率 | 日收益率标准差 × √252 | `std(daily_returns) × √252` | KPI卡片 |
| 最大回撤 | 净值从峰值到谷值的最大跌幅 | `max((cummax - nav) / cummax)` | KPI卡片 |
| 最大回撤峰值 | 回撤起始点的净值 | 回撤计算中对应峰值 | 详情卡片 |
| 最大回撤谷值 | 回撤结束点的净值 | 回撤计算中对应谷值 | 详情卡片 |
| 最大回撤起始日 | 回撤开始日期 | 回撤计算中对应日期 | 详情卡片 |
| 最大回撤结束日 | 回撤结束日期 | 回撤计算中对应日期 | 详情卡片 |
| VaR(95%) | 95%置信度下的日度在险价值 | `日收益率第5百分位数 × √252` | KPI卡片 |
| CVaR(95%) | 超越VaR的平均损失 | `E[损失 | 损失 > VaR] × √252` | KPI卡片 |
| 夏普比率 | 每承受1单位风险获得的超额收益 | `(年化收益 - 无风险利率) / 年化波动率` | KPI卡片 |
| Sortino比率 | 只考虑下行风险的夏普变体 | `(年化收益 - 无风险利率) / 下行标准差` | KPI卡片（如有数据） |
| Calmar比率 | 年化收益 / 最大回撤 | `年化收益 / 最大回撤` | KPI卡片（如有数据） |
| 最佳单日收益 | 日收益率最大值 | `max(daily_returns)` | KPI卡片 |
| 最差单日收益 | 日收益率最小值 | `min(daily_returns)` | KPI卡片 |
| Alpha | 超额收益（相对基准） | `empyrical.alpha(returns, benchmark)` | KPI卡片（如有基准） |
| Beta | 系统性风险暴露 | `empyrical.beta(returns, benchmark)` | KPI卡片（如有基准） |

### 4.1 大类资产穿透后分布

| 指标名称 | 指标说明 | 量化方法 | HTML输出方式 |
|----------|----------|----------|-------------|
| 权益类占比 | 穿透后权益资产占比 | 基金二级分类判定 + 基金内股票仓位加权 | ECharts饼图 |

**计算细节**（需求文档说明）:
